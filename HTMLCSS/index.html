HTML Nedir?

Merhaba !

HTML, internet sayfalarının yapı taşıdır. İnternet sayfalarını düşünürken bir hikaye kitabı veya bir poster yapımını hayal edebiliriz. HTML, bu hikaye kitabının veya posterin çerçevesini oluştururken içeriği ise metinler, resimler, bağlantılar gibi bileşenler oluşturur.

Web sayfaları, resimlerle renklendirilebilir. Bir resim, bir şeyi gösteren veya anlatan bir görseldir. Resimler, web sayfalarını daha ilgi çekici hale getirebilir.

Sayfalarda metinler kullanılır. Metinler, hikayeleri, bilgileri veya düşünceleri aktarmak için kullanılır. Metinler farklı boyutlarda, renklerde ve yazı tiplerinde olabilir.

Sayfalarda bağlantılar da bulunur. Bağlantılar, başka web sayfalarına veya farklı bölümlere yönlendiren tıklanabilir öğelerdir. Örneğin, bir bağlantıya tıkladığınızda farklı bir sayfaya geçebilirsiniz.

HTML, bu bileşenleri bir araya getirerek web sayfalarını oluşturur. Web sayfalarını tarayıcılarımız (örneğin Google Chrome veya Mozilla Firefox) okur ve bize gösterir. Web sayfalarının HTML kodlarını tarayıcılar anlar ve sayfayı bizim için doğru şekilde görüntüler.

HTML kullanarak, kendi hikaye kitaplarınızı veya posterlerinizi oluşturabilirsiniz. Metinleri düzenleyebilir, resimleri ekleyebilir ve bağlantılar oluşturabilirsiniz. Bu, web tasarımı ve internetin nasıl çalıştığı hakkında keyifli bir giriş olabilir.


CSS Nedir?

CSS, web sayfalarını renkli ve güzel hale getirmek için kullanılan bir araçtır. Web sayfalarını düşünürken, onları evlere benzetebiliriz. HTML evin çerçevesini oluştururken, CSS ona renk, şekil ve süslemeler ekler.

CSS ile web sayfalarındaki yazıları, başlıkları ve diğer öğeleri renklendirebiliriz. Ayrıca yazıların boyutunu, şekillerini ve yazı tiplerini de değiştirebiliriz. Örneğin, başlıkları sarı renkte veya büyük boyutta gösterebiliriz.

CSS, web sayfalarında farklı düzenlemeler yapmamızı sağlar. Sayfadaki öğeleri hizalayabilir, arka plan renklerini değiştirebilir ve hatta özel şekiller ekleyebiliriz. Örneğin, sayfanın arka planını mavileştirebilir veya resimlerin etrafına çerçeve ekleyebiliriz.

CSS, web sayfalarınızı kişiselleştirmenin harika bir yoludur. İster kendi blogunuzu tasarlayın, ister bir projeniz için bir web sayfası oluşturun, CSS size sayfanızı özelleştirmek için birçok seçenek sunar.

CSS kodlarını, web sayfalarının içine veya ayrı dosyalara ekleyebiliriz. Bu şekilde, sayfaların görünümünü kolayca değiştirebiliriz. CSS öğrenmek, size yaratıcı bir şekilde ifade etme ve web tasarımıyla ilgili ilginç şeyler yapma fırsatı verir.

CSS'yi kullanarak web sayfalarınızı renklendirip güzelleştirebilir, düzenleyebilir ve onlara kişisel bir tarz katabilirsiniz. Eğlenceli bir şekilde CSS öğrenerek, kendi web tasarım becerilerinizi geliştirebilirsiniz.

HTML ve CSS ile neler yapılabilir?

HTML ve CSS ile birçok şey yapılabilir. İşte HTML ve CSS ile yapılabilecek bazı şeyler:

1. Web Sayfaları Oluşturma: HTML ve CSS kullanarak kendi web sayfalarını oluşturabilirler. Başlıklar, paragraflar, resimler ve bağlantılar ekleyebilirler. Böylece ilgi çekici ve kişisel web sayfaları tasarlayabilirler.

2. Renkli ve Şekilli Metinler: CSS ile x metinleri renklendirilebilir, büyük veya küçük hale getirebilir ve hatta metinleri şekillendirebilir. Mesela, metni daire şeklinde veya kalp şeklinde gösterebilirler. Bu, metinleri eğlenceli hale getirir.

3. Resim Galerileri: HTML ve CSS kullanarak resim galerileri oluşturabilir. Birden fazla resmi bir araya getirebilir, resimlere başlık ekleyebilir ve resimleri yan yana veya üst üste yerleştirebilirler. Böylece kendi fotoğraf albümlerini oluşturabilirler.

4. Animasyonlar: CSS ile web sayfalarında basit animasyonlar yapılabilir. Örneğin, bir resmi hareket ettirebilir, renkli bir düğmeye tıkladıklarında arka planın değişmesini sağlayabilir veya bir metni yavaşça kaydırabilirler. Animasyonlar, web sayfalarını daha canlı ve eğlenceli hale getirir.

5. Basit Oyunlar: HTML ve CSS kullanarak basit oyunlar oluşturabilir. Örneğin, bir dizi sorunun olduğu bir quiz yapabilir veya bir kutucuğa tıkladıklarında puanlarını artıran bir hedef oyunu yapabilirler. Bu şekilde hem kodlama becerilerini kullanırken hem de eğlenirler.

Bu sadece HTML ve CSS ile yapılabilecek bazı örneklerdir. Birçok şey daha yapılabilir. HTML ve CSS öğrenmek, onlara dijital dünyada kendi fikirlerini ifade etme ve web tasarımıyla ilgili beceriler kazanma fırsatı sunar.

Neden HTML ve CSS Öğrenmeliyiz?

Şimdi sizlere neden HTML ve CSS öğrenmeniz gerektiğini anlatacağım

1. Kendi Web Sayfalarınızı Oluşturabilirsiniz: HTML ve CSS öğrenerek, kendi web sayfalarınızı tasarlayabilirsiniz. İster hikayelerinizi paylaşmak, ister bir projenizi tanıtmak veya hobilerinizi sergilemek için, kendi kişisel web sayfanızı oluşturabilirsiniz. Bu sayede, internet dünyasında kendinizi ifade etme fırsatı yakalarsınız.

2. Yaratıcılığınızı Geliştirebilirsiniz: HTML ve CSS öğrenmek, yaratıcılığınızı ve tasarım becerilerinizi geliştirmenize yardımcı olur. Web sayfalarınızı renklendirebilir, şekillendirebilir ve özelleştirebilirsiniz. Metinleri farklı boyutlarda ve renklerde görüntüleyebilir, resimleri ekleyebilir ve animasyonlar yapabilirsiniz. Bu sayede, kendi tarzınızı ve fikirlerinizi ifade etme şansınız olur.

3. İnternet Dünyasını Keşfedebilirsiniz: HTML ve CSS öğrenmek, internetin nasıl çalıştığını anlamanıza yardımcı olur. Web sayfalarının nasıl oluşturulduğunu ve tarayıcılarda nasıl görüntülendiğini öğrenirsiniz. Ayrıca, internet dünyasındaki diğer web sitelerini keşfedebilir ve farklı tasarımları inceleyebilirsiniz.

4. Kodlama Becerilerinizi Geliştirebilirsiniz: HTML ve CSS öğrenmek, temel kodlama becerileri kazanmanıza yardımcı olur. HTML, web sayfalarının yapısını oluştururken CSS, sayfaları stilize eder. Bu sayede, mantıklı düşünme, problem çözme ve kodlama yeteneklerinizi geliştirirsiniz. Ayrıca, diğer programlama dillerine geçiş yapma temelini oluşturur.

5. İletişim Becerilerinizi Geliştirirsiniz: Web sayfalarını oluştururken, düşüncelerinizi ve fikirlerinizi net bir şekilde ifade etmeniz gerekebilir. Sayfanızı düzenlemek, içeriği organize etmek ve ziyaretçilerle etkileşim kurmak için iletişim becerilerinizi kullanmanız gerekir. Bu da yazılı ve sözlü iletişim becerilerinizi geliştirmenize yardımcı olur.

HTML ve CSS öğrenmek, sizin için birçok fayda sağlar. Kendi web sayfalarınızı oluşturma, yaratıcılığınızı geliştirme, internet dünyasını keşfetme, kodlama becerilerinizi geliştirme ve iletişim becerilerinizi güçlendirme fırsatını yakalarsınız. Ayrıca, dijital dünyada kendinizi ifade etmek ve başkalarıyla paylaşmak için harika bir araç haline gelirsiniz.

HTML temelleri

HTML, internet sayfalarının yapı taşıdır. Bir web sayfasını oluştururken HTML kullanırız. HTML, etiketlerden oluşur. Her etiket, < işaretiyle başlar ve > işaretiyle biter. Etiketler, içerisine metin veya diğer öğeleri alabilir.

Birkaç temel HTML etiketinden bahsedelim

Başlık Etiketleri:

<h1>, <h2>, <h3>, <h4>, <h5>, <h6> gibi etiketler başlıkları belirtmek için kullanılır. <h1> en büyük başlığı temsil ederken, <h6> en küçük başlığı temsil eder.

Paragraf Etiketi:

<p> etiketi, paragrafları belirtmek için kullanılır. İçine yazdığınız metin, bir paragraf şeklinde görüntülenir.

Resim Etiketi:

<img> etiketi, bir resmi web sayfasına eklemek için kullanılır. src özelliği, resmin dosya yolunu veya internet üzerindeki URL'sini belirtir.

Bağlantı Etiketi:

<a> etiketi, bir bağlantı oluşturmak için kullanılır. href özelliği, bağlantının hedefini (bağlantıya tıklandığında nereye yönleneceğini) belirtir.

Bu etiketleri kullanarak, basit bir HTML sayfası oluşturabilirsiniz. Örneğin:

<!DOCTYPE html>
	<html>
		<head>
	  		<title>Benim İlk Web Sayfam</title>
		</head>
		<body>
	  		<h1>Merhaba Dünya!</h1>
	  		<p>HTML öğrenmek çok eğlenceli!</p>
	  		<img src="resim.jpg" alt="Harika Bir Resim">
	  		<a href="https://www.ornek.com">Daha Fazlası İçin Buraya Tıkla</a>
		</body>
	</html>

Bu şekilde HTML kullanarak basit bir websitesi yapabilirsiniz.

CSS temelleri



Renkleri Değiştirebiliriz: CSS ile metinlerin ve arka planın rengini değiştirebiliriz. İşte bir örnek: 

h1 {
	color: red;
  }
  
  p {
	color: blue;
  }

Bu kod, başlıkları kırmızı ve paragrafları mavi yapar.



Yazı Tipini ve Boyutunu Ayarlayabiliriz: CSS ile metinlerin yazı tipini ve boyutunu belirleyebiliriz. İşte bir örnek:

h1 {
	font-family: Arial, sans-serif;
	font-size: 24px;
  }

  p {
	font-family: Verdana, sans-serif;
	font-size: 16px;
  }

Bu kod, başlıkları Arial yazı tipiyle 24 piksel boyutunda ve paragrafları Verdana yazı tipiyle 16 piksel boyutunda yapar.



Kenarlık ve Arka Plan Ekleyebiliriz:

CSS ile sayfaya kenarlık ve arka plan özellikleri ekleyebiliriz. İşte bir örnek:

img {
	border: 2px solid black;
  }
  
  div {
	background-color: yellow;
  }
